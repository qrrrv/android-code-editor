# Инструкция по сборке APK

Этот проект настроен для автоматической сборки APK с помощью файла `build_apk.bat`.

## Требования для сборки на Windows:

1.  **Node.js & pnpm**: Установите [Node.js](https://nodejs.org/) и затем выполните `npm install -g pnpm`.
2.  **Java Development Kit (JDK)**: Рекомендуется JDK 17. Установите и настройте переменную окружения `JAVA_HOME`.
3.  **Android SDK**: Установите Android Studio и убедитесь, что установлены `Android SDK Platform` и `Build-Tools`. Настройте переменную окружения `ANDROID_HOME`.

## Как собрать APK:

1.  Внесите необходимые изменения в код.
2.  Настройте файл `.env` (скопируйте из `.env.example` и укажите адрес вашего сервера), иначе приложение не сможет подключиться к API.
3.  Запустите файл `build_apk.bat` двойным кликом.
4.  Дождитесь завершения процесса.
5.  Готовый файл появится в папке `builds/code-editor-debug.apk`.

## Что делает скрипт:

1.  Проверяет наличие `pnpm`.
2.  Устанавливает все зависимости проекта.
3.  Генерирует нативный Android-проект (папка `android`).
4.  Запускает сборку через Gradle.
5.  Копирует готовый APK в удобную папку `builds`.

---

**Примечание:** Если вы хотите собрать версию для публикации (Release), вам потребуется настроить ключи подписи (keystore) в файле `android/app/build.gradle`. По умолчанию скрипт собирает Debug-версию, которая отлично подходит для тестирования на телефоне.
